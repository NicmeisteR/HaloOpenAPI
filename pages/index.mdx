# HaloOpenAPI

**HaloOpenAPI** is community effort to open source and reverse engineer the Halo API.

Welcome to th Halo Open API docs, here you'll find in depth documentation on how the Halo Infinite (and eventually Halo MCC) API endpoints work by means of reverse engineering the internal Halo API's.

This website is built using NextJS, Nextra and MDX.
