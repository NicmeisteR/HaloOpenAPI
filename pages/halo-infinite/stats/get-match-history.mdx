# Get Match History:

Host: ```https://halostats.svc.halowaypoint.com```<br/>
Path: ```"/hi/players/{player}/matches"```<br/>
Type: ```GET```<br/>

### Headers: 
```json
{
    "accept": "application/json",
    "x-343-authorization-spartan": "v4="
}
```

## Params:
### count:
Specify the amount of matches to return.
* type: integer
* default: 25

### start:
Specify the offset/starting position of matches to return.
* type: integer
* default: 0

### include-times:
If true will return the times matches took place.
* type: boolean
* default: TBA
